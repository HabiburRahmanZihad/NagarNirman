# Security Policy

Currently, only the latest stable version of NagarNirman is supported with security updates.

| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of NagarNirman seriously. If you discover a security vulnerability within this project, please report it generally to us.

** Please do NOT report security vulnerabilities through public GitHub issues.**

### How to Report

Please email the detailed vulnerability report to:
✉️ **security@nagarnirman.com** (or the project maintainer email)

### What to Expect

- We will acknowledge receipt of your vulnerability report within **48 hours**.
- We will send you a detailed confirmation once the issue has been verified.
- We will notify you when a fix is deployed.
